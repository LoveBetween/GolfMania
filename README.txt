Lien git vers les sources du projet Unity:

https://github.com/LoveBetween/GolfMania.git

Il contient 3 scènes : la scène "Start Menu" qui contient le menu principal scène "Golfmania" qui contient la démo et la scène "SampleScene" qui contient le club avec les flexpoints.

Pour chaque scène, c'est possible d'appuie sur la touche "échap" pour revenir au menu principal.

Start Menu:

Avec la souris, vous pouvez démarrer la partie en appuyant sur -Play (scène GolfMania)
							       -Soft golf club (scène Sample Scene)
                                                               -Settings (fonctionnalité non implémenter)
                                                               -Quit (quitter le jeu)

GolfMania:

Pour pouvoir tester la scène il faut disposer d'un casque Meta Quest, en utilisant la manette droite vous pouvez tirer la balle de golf avec le club.


Sample Scene:

Pour afficher le club flexible, il suffit de lancer la scène.

Le fichier "Clubs/Fer 7" contient le Scriptable Object permettant de fixer le nombre de flexpoints souhaité.

Au lancement, comme nous n'avons plus la fonctionnalité VR, il faut manuellement modifier les valeurs de position dans le tableau flex_point de l'inspecteur à droite.
